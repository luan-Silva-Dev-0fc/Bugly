## 🚀 AI Terminal Error Analyzer

Bugly PRO é uma ferramenta inteligente para análise automática de erros
de terminal e logs de programação.

Ele identifica automaticamente:

-   📂 Caminho da pasta com erro\
-   📄 Arquivo afetado\
-   📍 Linha exata do problema\
-   🔴 Nível de severidade\
-   📦 Possível dependência faltante\
-   🛠 Passos claros para correção

Interface moderna, dark, com visual tecnológico e sistema inteligente de
análise.

## 🔑 Configuração da API

No arquivo `script.js`, configure sua chave:

``` javascript
const API_KEY = "SUA_API_AQUI";
```

⚠️ Para produção, recomenda-se mover a API para um backend.

------------------------------------------------------------------------

## 🧠 Como Funciona

1.  Usuário cola um erro de terminal.
2.  O sistema valida e detecta automaticamente:
    -   Caminho do arquivo
    -   Linha do erro
    -   Severidade
3.  Envia para a IA.
4.  Retorna diagnóstico estruturado + mapa visual.
